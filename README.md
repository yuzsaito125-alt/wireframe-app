# Wireframe Animation App

## 初回セットアップ

```cmd
cd /d "e:\@YuzuruSaito\仕事フォルダ\@kiro\wireframe-app"
python -m pip install flask qrcode pillow
```

## 起動

```cmd
cd /d "e:\@YuzuruSaito\仕事フォルダ\@kiro\wireframe-app"
python app.py
```

ブラウザで http://localhost:5000 を開く。

## スマホからアクセス

1. ページの「📱 QR」ボタンを押す
2. スマホのカメラでQRコードをスキャン
3. スマホとPCが同じWi-Fiに繋がっていること

## 停止

コマンドプロンプトで `Ctrl + C`
